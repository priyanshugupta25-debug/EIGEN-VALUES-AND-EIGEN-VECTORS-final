üìòEigen Value & Eigen Vector

Eigenvalues and eigenvectors are among the most important concepts in linear algebra. They provide deep insights into the behavior of linear transformations and matrices, revealing the directions along which a transformation acts by simple scaling.

The word "eigen" comes from German, meaning "own" or "characteristic." An eigenvector of a matrix is a non-zero vector that, when the matrix is applied to it, only changes in scale (magnitude) but not in direction.

  • Physics: Vibration analysis, quantum mechanics, and stability analysis
  • Data Science: Principal Component Analysis (PCA) for dimensionality reduction
  • Engineering: Structural analysis and control systems
  • Computer Graphics: Image compression and face recognition
  • Google PageRank: Ranking web pages using the dominant eigenvector

üìò What are Eigenvalues?

DEF Eigenvalue

A scalar Œª (lambda) is called an eigenvalue of a square matrix A if there exists a non-zero vector v such that:

Av = Œªv

In simpler terms, an eigenvalue is a special number associated with a matrix. When you multiply the matrix by its eigenvector, the result is the same as multiplying the eigenvector by this eigenvalue.

Think of a matrix as a transformation that stretches, rotates, or reflects vectors. An eigenvalue tells you how much a vector is stretched (or shrunk) along a particular direction.

  • Œª > 1: The vector is stretched
  • 0 < Œª < 1: The vector is compressed
  • Œª = 1: The vector length is unchanged
  • Œª < 0: The vector is flipped and scaled
  • Œª = 0: The vector is collapsed to the origin

üìò What are Eigenvectors?

DEF Eigenvector

A non-zero vector v is called an eigenvector of matrix A corresponding to eigenvalue Œª if:

Av = Œªv

An eigenvector is a special direction in which the linear transformation represented by the matrix acts by pure scaling. The vector may get longer, shorter, or flip direction, but it stays on the same line through the origin.
